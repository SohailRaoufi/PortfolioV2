# Terminal Portfolio
Minimal terminal-style portfolio built with Vite + TypeScript.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Edit Content
- Update personal data, projects, and colors in `config.json`.
- Update editable command content in `config.json` under `terminalContent`.
- Command behavior and routing live in `src/main.ts` and `src/commands/`.
